Version control system

version control system is also know as source control, is the practice of tracking and managing changes to software code. is a software tools that help software team manage changes to source code over time. This help teams stores and collaborate while writting software applications, it is use for tracking every individual changes by each contribution and help to prevent concurrent work from conflicting, is use to disable some portion of a code, so a computer won't process it, but the code won't be deleted because is still useful for later date.

Git

Git is a type of version control system, designed to handle everything from small to very large projects with speed and efficiency. is a free and open source version control system. it's light weight and very fast, it has an excellent support for branching, merging and rewriting history.

Process of commiting a projects to github repository

  1. first create a repository in your github account
  2. open your vsc terminal where you have the project you want to commit
  3. type git init
  4. git add .
  5. git commit -m "eg my second commit"
  6. git branch -M main
  7. git remote add origin the repository link from your remote repository in your github
  8. git push -u origin main
